# FFmpeg2AndroidDemo
## 内容包含:
> * FFmpeg 编译脚本
> * libx264静态库
> * libfdk-aac静态库
> * cMake编译FFmpeg为可命令执行的脚本
> * Android端命令执行FFmpeg的接口
> * Android使用案例
